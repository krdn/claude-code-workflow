## 💬 도움 받기

### 1. GitHub Discussions (권장)

일반적인 질문, 아이디어 공유, 토론에 가장 적합합니다.

**[→ Discussions 바로가기](https://github.com/krdn/claude-code-workflow/discussions)**

| 카테고리 | 용도 |
|----------|------|
| 💡 Ideas | 새로운 아이디어 제안 |
| 🙏 Q&A | 질문 및 답변 |
| 🙌 Show and tell | 자신의 워크플로 공유 |
| 💬 General | 일반 토론 |

#### 좋은 질문 예시

```markdown
제목: MCP 서버를 여러 개 동시에 설정하는 방법

안녕하세요!

GitHub MCP와 Postgres MCP를 함께 사용하고 싶습니다.
현재 examples/mcp/README.md를 참고했는데,
단일 서버 설정 예제만 있어서 질문드립니다.

**시도해본 것:**
- config.json에 두 서버를 나열해봤지만 에러 발생

**환경:**
- OS: macOS 14.0
- Claude Code: 1.0.0

도움 부탁드립니다!
```

## 🔍 자주 묻는 질문 (FAQ)

### Q: Claude Code가 뭔가요?

Claude Code는 Anthropic에서 개발한 AI 기반 코딩 어시스턴트입니다. 이 저장소는 Claude Code를 효과적으로 사용하기 위한 워크플로와 모범 사례를 제공합니다.

### Q: 이 저장소는 공식 저장소인가요?

아니요, 이 저장소는 커뮤니티에서 운영하는 비공식 저장소입니다. 공식 문서는 [Anthropic 공식 문서](https://docs.anthropic.com/claude-code)를 참조하세요.

### Q: 기여하고 싶은데 어떻게 하나요?

[CONTRIBUTING.md](CONTRIBUTING.md)를 참조해 주세요. 처음 기여하시는 분은 `good first issue` 라벨이 붙은 이슈부터 시작하시는 것을 권장합니다.

### Q: 버그를 발견했어요!

[Issue 생성](https://github.com/krdn/claude-code-workflow/issues/new?template=bug_report.yml)을 통해 신고해 주세요. 보안 관련 버그는 [SECURITY.md](SECURITY.md)를 참조해 주세요.
